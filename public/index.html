Real-time AFINN-based sentiment analysis on streaming Twitter feed

Twitter Streaming feed is requested based on user-chosen keywords and then result is visualized by change in background color based on real time sentiment analysis on keyword. Average sentiment is calculated by total number of tweets processed in real time divided by sum of estimate of each tweet sentiment made from its text.

Technologies Used:

  • NODEJS
  • ExpressJS
  • Twitter APIs
  • Socket.io
  • HTML
